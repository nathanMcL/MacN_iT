Understanding the For Loop in Detail

Diving into Loops, July 3, 2024

07/03/2024 For Loop Deconstructed

            
            for (initialExpression; conditionExpression; updateExpression) {
               // Loop body
            }
  • initialExpression...
    The initialExpression is executed at the beginning of the loop, and is used as a loop control variable.
    int i = 0; initializes the loop variable i to 0...
  • conditionExpression...
    The conditionExpression is evaluated before each iteration of the loop. If the condition is true, the loop body is executed; else, if false, the loop terminates.
    If i < 10, then the loop continues if i is less than 10...
    If arr[i] != -1, then the loop continues as long as the element at the index i of array arr is not equal to -1.
  • updateExpression...
    The updateExpression is executed after each iteration of the loop body, and is used to update the initialExpression variable.
    i++ increases the initialExpression value by 1...
    i += 2 increments the initialExpression value by 2...
    i-- decreases the initialExpression value by 1...

An Example of a for Loop:

            
            for (int i = 0; i < 10; i++) {
                printf("%d ", i);
            }
            printf("\n");
